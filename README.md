## 项目要求概述
    一个功能完整的个人博客系统，该系统包含用户注册、登录、博客发布、博客浏览、评论互动等基本功能。运用Servlet、JSP、JDBC、过滤器、监听器等技术实现该项目。
## 功能需求

 - 用户管理模块
 - 用户注册（用户名、密码、邮箱）
 - 用户登录/登出
 - 用户信息查看和修改
 - 博客管理模块
 - 发布新博客（标题、内容、分类）
 - 浏览博客列表（分页显示）
 - 查看博客详情
 - 编辑和删除自己的博客
## 评论管理模块
   - 对博客进行评论
   - 查看博客的评论列表
   - 其他功能
## 博客分类管理
   - 博客点击量统计
   - 分页功能
## 技术要求
### Servlet技术：
 - 5个Servlet（登录、注册、发布博客、查看博客列表、处理评论等）
 - 使用Request和Response对象
 - 合理使用Session管理用户状态
 - 使用Application对象存储全局信息
### JSP技术：
   - 5个JSP页面（首页、登录、注册、发布博客、博客详情等）
   - 使用JSTL标签库
   - 使用EL表达式
   - 使用JSP内置对象
### 数据库操作：
   - 使用JDBC连接数据库
   - 实现数据库的增删改查操作
   - 使用PreparedStatement防止SQL注入
